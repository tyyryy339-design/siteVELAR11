# 🚗 VELAR - Аренда автомобилей

## ⚡ Быстрый старт

### 🖥️ На компьютере:

1. **Запусти сервер:**
   - Двойной клик на файл `start-server.bat`
   - Откроется окно с сервером

2. **Открой сайт:**
   - В браузере: `http://localhost:8080/index.html`

### 📱 На телефоне:

1. **Быстрый доступ с QR:**
   - На компьютере откройте: `http://localhost:8080/phone.html`
   - Отсканируйте QR код телефоном
   - Готово!

2. **Или вручную:**
   - Убедись, что телефон в одной WiFi с компьютером
   - В браузере телефона вбей: `http://[IP_КОМПЬЮТЕРА]:8080`
   - Например: `http://192.168.1.100:8080`

## 🔧 Как добавить машину с фото?

1. Откройте `manage-cars.html`
2. Заполните форму и загрузите фото
3. Зарегистрируйте данные автомобиля
4. Фото автоматически появится на сайте
